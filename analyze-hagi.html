How to Backfill Missing Data

Use the CLI scripts to identify and backfill missing dates:

  1. Diagnose all gaps: node scripts/diagnose-data-gaps.js
  2. Get missing dates: node scripts/backfill-checker.js 2025-01-01 2025-01-31
  3. Pipe to scraper: node scripts/backfill-checker.js 2025-01-01 2025-01-31 | xargs -I{} your-scraper --date {}
  4. Regenerate mii_results_latest.csv and upload both CSVs to S3.
